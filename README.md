# 📊 Digital User Behavior Analysis Dashboard

## 📌 Project Overview

This project was completed as part of the **LogicStack Data Analysis Internship (July 2026)**.

The objective of this project is to analyze user behavior, understand the customer conversion funnel, identify drop-off points, evaluate revenue performance, and build an interactive Power BI dashboard for business decision-making.

---

## 🎯 Project Objectives

* Analyze user behavior using SQL
* Understand the customer conversion funnel
* Identify drop-off points
* Analyze revenue performance
* Build an interactive Power BI dashboard
* Generate business insights and recommendations

---

## 🛠 Tools Used

* MySQL
* Power BI
* Microsoft Excel

* User ID

The following analyses were performed:

## 📊 Power BI Dashboard

The dashboard includes:

* KPI Cards
* User Journey Funnel
* Revenue by Channel
* Revenue by Region
* Device Distribution
* Revenue Trend
* Insight Panel

---

## 🔍 Key Insights

* The largest user drop-off occurs between Browse and Add to Cart.
* Google Ads is the highest-performing marketing channel.
* South region generates the highest revenue.
* Desktop users have the highest conversion rate.
* The purchase conversion rate indicates opportunities for funnel optimization.

---

## 💡 Business Recommendations

* Improve the Browse to Add to Cart experience.
* Increase investment in Google Ads.
* Optimize the mobile user experience to improve conversions.
